Definition of a Limit

A limit of a function is the value it approaches at that point:


lim x → c ⁡ f ( x )

A limit can be either approached from the left or the right, and the limit only exists if from both sides, it approaches the same value:


lim x → c − ⁡ f ( x ) = lim x → c + ⁡ f ( x ) = k

Continuity

A function f is continuous at x=c if and only if:

lim x → c ⁡ f ( x ) exists
f ( c ) exists
lim x → c ⁡ f ( x ) = f ( c )

Differentiability

A function is differentiable at c if it is smooth (no cusps nor corners) and continuous.

Evaluating Limits

  1. Direct Substitution (Treat it like a regular function).
  2. If the limit has a removable discontinuity, attempt to factor out and divide.
  3. If it applies, use L'hospital's rule.

Squeeze Theorem

If f ( x ) ≤ g ( x ) ≤ h ( x ) near x = a and lim x → a ⁡ f ( x ) = lim x → a ⁡ h ( x ) = L then lim x → a ⁡ g ( x ) = L

Limits and Infinity

Limits can be evaluated at infinity, which means we need to look at the behavior of the function as the variable increases/decreases non-stop.


  1. Limits may evaluate to a specific value:
    lim x → ∞ ⁡ 1 x = 0
  2. Limits may diverge, or approach infinity:
    lim x → 0 ⁡ 1 x = ∞
  3. Or may oscillate/not exist:
    lim x → ∞ ⁡ s i n ( x ) = D N E

A good way to evaluate the type of limits that appear as f(x)/g(x) as x→∞ is to look at the 'growth' of the functions.

Factorial >> Exponential >> Polynomial >> Logarithmic >> Trigonometric

  • If f(x) 'grows' faster than g(x), the limit must approach infinity.
  • If g(x) 'grows' faster than f(x), the limit must approach zero.
  • Polynomials of the same degree have the same 'growth' rate, and if both f(x) & g(x) are polynomials of the same degree, then take the first coeficients as a fraction.
